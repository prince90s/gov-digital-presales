# 标前检查清单

> 投标前最后一道防线。两人独立交叉检查，逐项打勾确认。

---

## 一、资质文件检查

| # | 检查项 | 要求说明 | ✓ |
|---|--------|---------|---|
| 1.1 | 营业执照副本 | 有效期内，经营范围覆盖投标内容 | |
| 1.2 | CMMI / ISO9001 / ISO27001 | 有效期内，附年审记录 | |
| 1.3 | 高新技术企业证书 | 政府项目普遍认可 | |
| 1.4 | 涉密信息系统集成资质 | 涉及涉密数据时必须具备 | |
| 1.5 | 项目经理资质 | 软考高级/中级 + 证书 + 社保 | |
| 1.6 | 项目团队社保 | 投标截止日前连续3个月社保证明 | |
| 1.7 | 无重大违法记录声明 | 按招标文件格式，加盖公章 | |
| 1.8 | 信用中国报告 | 投标截止日前7日内下载 | |

**常见废标点**：证书过期未附年审 → 直接废标；社保证明时间不满足 → 直接废标

---

## 二、技术标检查

| # | 检查项 | 要求说明 | ✓ |
|---|--------|---------|---|
| 2.1 | 目录与页码 | 正文页码与目录完全一致 | |
| 2.2 | 技术参数响应表 | 逐条点对点响应；"完全响应"须有依据 | |
| 2.3 | 技术偏离表 | 正偏离须有证明材料 | |
| 2.4 | 项目实施方案 | 流程图、甘特图、人员安排表齐全 | |
| 2.5 | 项目团队 | 简历+证书+社保+无在建承诺齐全 | |
| 2.6 | 等保/密评方案 | 独立章节，等保三级逐条响应 | |
| 2.7 | 信创适配方案 | CPU/OS/数据库/中间件国产化计划明确 | |
| 2.8 | 典型案例 | 合同+验收报告复印件齐全，相似度高 | |
| 2.9 | 图纸/拓扑图 | 所有图片清晰，编号连续 | |

---

## 三、商务标检查

| # | 检查项 | 要求说明 | ✓ |
|---|--------|---------|---|
| 3.1 | 投标函 | 按标书格式，签字盖章齐全 | |
| 3.2 | 投标报价表 | 大小写金额一致；分项报价之和=总报价 | |
| 3.3 | 投标保证金 | 到账凭证附在标书中 | |
| 3.4 | 法定代表人授权书 | 格式符合标书要求，附双方身份证复印件 | |
| 3.5 | 中小企业声明函 | 如适用，按工信部标准格式 | |

**常见废标点**：报价超过最高限价 → 废标；投标函未签字盖章 → 废标

---

## 四、格式与装订检查

| # | 检查项 | 要求说明 | ✓ |
|---|--------|---------|---|
| 4.1 | 文件份数 | 正本X份、副本X份，按标书要求 | |
| 4.2 | 装订方式 | 胶装/线装，按标书要求（通常胶装，不可活页） | |
| 4.3 | 密封要求 | 按标书要求密封 | |
| 4.4 | 电子版文件 | 是否需要U盘/光盘，格式要求（PDF/Word） | |
| 4.5 | 签字盖章 | 逐页检查：是否需要逐页小签 | |
| 4.6 | 电子投标 | CA锁是否有效；文件加密/签名成功；上传完成 | |

---

## 五、电子投标特别提醒

| # | 检查项 | 说明 |
|---|--------|------|
| 5.1 | CA锁有效期 | 提前检查，避免开标当天过期 |
| 5.2 | 电子标书生成 | 使用招标方指定工具，检查生成日志 |
| 5.3 | 上传完整性 | 上传后下载回验 |
| 5.4 | 开标电脑环境 | 提前测试：CA驱动、浏览器插件、网络 |
| 5.5 | 开标解密 | 开标前30分钟上线，准备好CA密码 |
